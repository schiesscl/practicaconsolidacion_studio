Projects

Task Managing Application ("TaskMaster")

TaskMaster is a web application that allows users to manage their daily tasks efficiently.

Users can create, update and delete tasks, as well as classify them into different categories. User authentication, advanced filters and push notifications have been implemented to remind users about pending tasks.

The application is built with Node.js and Express on the backend, and React on the frontend.

Online Course Platform ("EduLearn")

EduLearn is a web platform that offers online courses in various fields.

Users can register, enrol in courses, complete lessons and take exams. Administrators can add and manage courses, as well as view user participation statistics.

The platform is developed with a MEAN stack (MongoDB, Express, Angular, Node.js) and uses JWT for user authentication.

Culinary Recipe Application ("CookBook")

CookBook is a web application where users can search, save and share recipes.

Users can filter recipes by ingredients, category and difficulty. The application allows registered users to create and manage their own recipes, including photos and videos.

CookBook is built on a MERN architecture (MongoDB, Express, React, Node.js) and uses GraphQL for efficient queries.
